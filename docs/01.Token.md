## クラス明細

### `NumberToken`

#### `GetValue()` 処理フロー
```mermaid
flowchart TD
    s[start]
    C1[文字領域: charactor]
    C2[数値領域: result=0]
    C2[数値領域: digit=0]
    StartMainLoop[/文字列の終端まで\]
    F4[charactorに次の文字を入れる]
    F1[resultを10倍にする]
    F2[digit = charactorの文字コードから'0'も文字コードを引く]
    F3[resultにdigitを足す]
    EndMainLoop[\ /]
    e([end])

    s-->C1-->C2-->StartMainLoop
    StartMainLoop-->F1-->F2-->F3-->F4-->EndMainLoop
    EndMainLoop-->e
    
```

### `OperatorToken`

#### `GetValue()` デシジョンテーブル

| `LeftHand` | `Kind`     | `RightHand` | 処理                    |
| ---------- | ---------- | ----------- | ----------------------- |
| null       | Any        | null        | Error                   |
| _          | Any        | null        | Error                   |
| null       | Plus/Minus | _           | RightHandに符号をつける |
| null       | Other      | _           | Error                   |
| _          | Any        | _           | 演算処理をする          |
